# recipes_app
This project is a Recipe Application that allows users to explore various food recipes and also add their own custom recipes. The app provides a smooth and modern user experience with support for both Dark Mode and Light Mode.
Features:
Authentication: Login, Signup, Logout
View Recipes: Browse a large collection of food recipes
Favorites: Save and manage your favorite recipes
Search: Quickly search for any recipe
Add Recipes: Create your own recipe
Edit & Delete Recipes: Full control over user-created content
Dark & Light Mode support
Feature-Based Project Structure
MVVM Architecture with Cubit state management
Technologies & Packages Used:
flutter_secure_storage — Secure local data storage
shared_preferences — Lightweight local storage
json_serializable — Automatic model generation
retrofit — Networking & API handling
Cubit (flutter_bloc) — State management
Architecture:
The project follows the MVVM (Model–View–ViewModel) architecture combined with a Feature-Based structure, ensuring clean, scalable, and maintainable code.
